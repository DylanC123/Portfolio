Photoline - GreatUniHack 2024

  • An application made using React, along with PrimeReact, for the front-end and Go for the back-end. Photoline is an application which helps users capture memories during their life, in the same way as a photo album. Users can add 'memories' which are added to their timeline as a node with a name and date. The user can then click on the memory to expand it and see a description and any photos which have been uploaded to that specific memory. However, often memories involve other people, so it has a friend feature. Each user has a unique code to be used for adding other users. When creating a memory the user can add selected friends to it. This memory will then appear on their timeline as well, allowing them to also add photos. Another small but very key feature of having private images as a user may want some photos to be added to the memory but not be seen by other users as it is personal.

Evolution Simulation - A level coursework mark 70/70

  • Made in Unity with C++ for A-level coursework, achieving full marks. Involved procedural generation of a random 3D map with water and terrain. It involved the life cycle of Rabbits with basic attributes - eating, drinking and reproduction - which evolved over time. This is a very basic form of AI, representing 'Survival of the Fittest', showing a real life example of a genetic algorithm.
